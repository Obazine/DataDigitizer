Data Digitizer Dataset Guide

  • PLEASE TAKE NOTE THAT YOU MUST BE LOGGED IN TO USE DATASET FEATURES
  • 1. Before you do anythin, you must first create a dataset. This will reset your current temporary graph so make sure to save the image before hand.
  • 2. If you ever want to switch between your datasets, you can choose "Select Dataset" and once selecting your desired datasets, all the values regarding the graph will be loaded to your session
  • 3. If you want to clean your workspace, or decide your dataset is no longer needed, you can delete them by going to the "Delete Dataset" section and select the database from a drop down list, similar to the select dataset method.
  • 4. If you are using manual points extraction, when you want to export your dataset to a CSV file, you must click "Export Dataset" to export the data points.
